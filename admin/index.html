💡 Como configurar a atualização automática:
  • No Netlify: Vá em Site settings → Build & deploy → Build hooks → clique em Add build hook (nomeie como "CMS") e copie a URL gerada.
  • Na Vercel: Vá em Project Settings → Git → role até Deploy Hooks → digite um nome e crie, depois copie a URL gerada.
